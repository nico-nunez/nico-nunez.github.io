meh-groovebox-visceral/

meh-groovebox-visceral

A code-driven performance instrument for macOS, built from scratch in C/C++. The synth engine is a separate extracted library running 64-voice polyphony with 4 wavetable oscillators per voice, a 16-slot modulation matrix with FM routing, configurable per-voice filter chains, and a full post-processing stack (chorus, phaser, delay, reverb). The groovebox host wraps it with a step sequencer with per-step parameter locks, multi-track mixing, and a Lua REPL with a built-in editor and language server for live configuration. No traditional GUI yet, though that's still on the table depending on how the Lua approach plays out.

  • Lua
  • CoreAudio
  • CoreMIDI
